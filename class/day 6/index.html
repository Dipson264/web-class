CSS Units

  1. px(pixels) -> fixed (not used for responsive site)
  2. rem ->relative to the root element (HTML) ->16px
  3. em ->relative to parent element
  4. % ->relative to parent element
  5. vh -> relative to viewport's height
  6. vw -> relative to viewport's width
